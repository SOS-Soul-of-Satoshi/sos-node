# Security Policy

## Supported Version

Only the latest public testnet release is supported. Testnet tokens have no
monetary value, and this software has not completed an independent audit.

## Reporting A Vulnerability

Do not open a public issue for a suspected vulnerability. Use GitHub's private
vulnerability reporting form:

https://github.com/SOS-Soul-of-Satoshi/sos-node/security/advisories/new

Include the affected release or commit, component, impact, reproduction steps,
and any proposed mitigation. Do not include production credentials or private
keys. Reports are acknowledged and triaged on a best-effort basis; no bug bounty
or response-time guarantee is currently offered.
